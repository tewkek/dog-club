# dog-club
в качестве бд для проекта использовался Postgresql, при первом выполнении кода при подключении к postgre будет создана бд dogclub, для использования модели из roboflow необходимо будет добавить ее в бд (напрямую через pgAdmin4 или через админ панель веб приложения), введя название, тип модели (онлай или локально), ключ API (пример : "sBpmsYXLwuSUxULiR8b4" можно получить на странице модели на сайте roboflow), ссылку на модель (пример "dog-3lbpq/1", также находиться на странице модели в roboflow), введя точность модели, и выбора ее в активный слот
Для запуска проекта, можно использовать Visual Studio Code, введя в bash терминале npm start, после чего запуститься браузер по умолчанию, с запущенный веб приложением
